# 我的筆記

這裡是 `notes.md` —— MindVault 的主筆記檔。用 Markdown 的 `## 標題` 分類、`- 項目` 列點就會自動被拆成區塊和條目。

## 🎯 目標

- 這是範例內容。把檔案換成你自己的筆記即可。
- 在 `.env` 設 `DATA_DIR=/path/to/your/notes` 可改指到任何資料夾。
- 推薦做法：把 `DATA_DIR` 指向一個獨立的 git repo，見 `docs/data-management.md`。

## 💡 靈感

- 專案靈感來自 [Andrej Karpathy 的 LLM Wiki 概念](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)：raw sources 不變、LLM 維護一層 wiki 作為 synthesis layer。

# MindVault üßÝ

AI-powered personal knowledge base ‚Äî browse notes, documents, and memories through a clean web interface with Gemini-powered summarization and categorization.

**Live:** https://lewsi.ddns.net/mind-vault/

---

## What It Does

- **Overview** ‚Äî Stats dashboard (doc count, word count, notes items) + recently updated docs, all clickable
- **Library** ‚Äî All documents grouped by folder, with inline markdown reader and AI summary per doc
- **Categories** ‚Äî Gemini auto-categorizes all notes and documents into smart groups
- **Search** ‚Äî Full-text search across notes and all documents
- **Raw Notes** ‚Äî Rendered markdown view of `notes.md`

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/notes` | Parse notes.md into sections + items |
| GET | `/api/library` | All docs with metadata, preview, mtime |
| GET | `/api/stats` | Total docs, words, notes items, folder counts |
| GET | `/api/doc?folder=&name=` | Full document content |
| GET | `/api/summary` | AI summary of entire knowledge base (cached 1h) |
| GET | `/api/categorize` | AI categorization of all notes + docs (cached 1h) |
| GET | `/api/doc-summary?folder=&name=` | AI summary of a single doc (cached 1h) |
| GET | `/api/search?q=` | Full-text search across notes and docs |
| GET | `/api/health` | Health check |

## Data Sync

Documents are synced from `/workspace/group/` on the nano agent:

- `notes.md` ‚Üí `data/notes.md`
- `articles/*.md` ‚Üí `data/articles/`
- `saves/*.md` ‚Üí `data/saves/`
- `conversations/*.md` ‚Üí `data/conversations/`

A daily cron task at 2am syncs new files automatically. New files added to workspace are also manually synced by nano.

## Deployment

### First deploy
The GitHub Actions workflow auto-creates:
- systemd service at `/etc/systemd/system/mind-vault.service`
- nginx config at `/etc/nginx/conf.d/projects/mind-vault.conf`

### Subsequent deploys
Push to `main` ‚Üí GitHub Actions builds frontend ‚Üí rsyncs to VPS ‚Üí restarts service.

### Required GitHub Secrets
| Secret | Value |
|--------|-------|
| `VPS_HOST` | `lewsi.ddns.net` |
| `VPS_USER` | `lewsi` |
| `VPS_SSH_KEY` | ED25519 private key |
| `GEMINI_API_KEY` | Google AI Studio key |

### VPS Details
- **Port:** 10016
- **Path:** `/srv/projects/mind-vault/`
- **Service:** `systemctl status mind-vault`
- **Logs:** `journalctl -u mind-vault -f`

## Local Dev

```bash
# Backend
cd MindVault
uv sync
uv run uvicorn src.main:app --reload --port 10016

# Frontend (separate terminal)
cd frontend
npm install
npm run dev   # proxies /api/* to localhost:10016
```

Frontend dev server runs on `http://localhost:5173` with API proxy configured in `vite.config.ts`.

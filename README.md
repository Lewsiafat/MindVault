# MindVault 🧠

AI-powered personal knowledge base — browse notes, documents, and memories through a clean web interface with Gemini-powered summarization and categorization.

**Live:** https://lewsi.ddns.net/mind-vault/

---

## What It Does

- **Overview** — Stats dashboard (doc count, word count, notes items) + recently updated docs, all clickable
- **Library** — All documents grouped by folder, with inline markdown reader and AI summary per doc
- **Categories** — Gemini auto-categorizes all notes and documents into smart groups
- **Search** — Full-text search across notes and all documents
- **Raw Notes** — Rendered markdown view of `notes.md`
- **Wiki** — Per-document AI summarization, concept ingestion, and cross-document knowledge synthesis
- **Lint** — AI health-check of notes quality with severity scoring and one-click fix
- **RWD** — Fully responsive; hamburger slide-in sidebar on mobile (≤768px)

---

## Stack

| Layer | Tech |
|-------|------|
| Backend | FastAPI + uvicorn (Python 3.13) |
| Frontend | Vue 3 + TypeScript + Vite |
| AI | Google Gemini 2.0 Flash (`google-genai` SDK) |
| Deploy | GitHub Actions → rsync → VPS systemd |
| Reverse proxy | nginx |

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/notes` | Parse notes.md into sections + items |
| GET | `/api/library` | All docs with metadata, preview, mtime |
| GET | `/api/stats` | Total docs, words, notes items, folder counts |
| GET | `/api/doc?folder=&name=` | Full document content |
| GET | `/api/summary` | AI summary of entire knowledge base (cached 1h) |
| GET | `/api/categorize` | AI categorization of all notes + docs (cached 1h) |
| GET | `/api/doc-summary?folder=&name=` | AI summary of a single doc (cached 1h) |
| GET | `/api/search?q=` | Full-text search across notes and docs |
| GET | `/api/wiki/status` | Wiki build status and pending docs |
| GET | `/api/wiki/pages` | All wiki pages |
| POST | `/api/wiki/ingest` | Ingest a document into wiki |
| POST | `/api/wiki/synthesize` | Cross-document knowledge synthesis |
| POST | `/api/lint` | Run AI lint health-check on notes |
| POST | `/api/lint/fix` | Auto-fix a single lint issue |
| GET | `/api/health` | Health check |

## Data Sync

Documents are synced from `/workspace/group/` on the nano agent:
- `notes.md` → `data/notes.md`
- `articles/*.md` → `data/articles/`
- `saves/*.md` → `data/saves/`
- `conversations/*.md` → `data/conversations/`

A daily cron task at 2am syncs new files automatically.

---

## Deployment

Push to `main` → GitHub Actions builds frontend → rsyncs to VPS → restarts systemd service.

- **Port:** 10016
- **Path:** `/srv/projects/mind-vault/`
- **Logs:** `journalctl -u mind-vault -f`

---

## Local Dev

```bash
uv sync && uv run uvicorn src.main:app --reload --port 10016
cd frontend && npm install && npm run dev
```
